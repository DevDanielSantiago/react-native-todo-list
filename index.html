Aplicativo ToDo

Um aplicativo móvel de gerenciamento de tarefas construído com React Native, Expo, TypeScript e Zustand.

Funcionalidades

  • Criar, atualizar e deletar tarefas
  • Marcar tarefas como concluídas
  • Contador de tarefas criadas e concluídas
  • Interface limpa e intuitiva
  • Validação de entrada
  • Persistência de dados com Zustand

Gerenciamento de Estado

O aplicativo usa Zustand para gerenciamento de estado. A store é configurada para lidar com:

  • Gerenciamento da lista de tarefas
  • Atualizações de status das tarefas
  • Persistência de dados

Exemplo de uso da store:

import { create } from 'zustand';
 
 interface TaskStore {
   tasks: Task[];
   addTask: (task: Task) => void;
   toggleTask: (id: number) => void;
   removeTask: (id: number) => void;
 }
 
 const useStore = create<TaskStore>((set) => ({
   tasks: [],
   addTask: (task) => set((state) => ({ tasks: [...state.tasks, task] })),
   toggleTask: (id) => set((state) => ({
     tasks: state.tasks.map(task => 
       task.id === id ? { ...task, done: !task.done } : task
     )
   })),
   removeTask: (id) => set((state) => ({
     tasks: state.tasks.filter(task => task.id !== id)
   })),
 }));
